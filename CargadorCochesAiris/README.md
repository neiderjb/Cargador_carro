# Cargador de Coches



## Overview

This firmware Charging Vehicule to AIRIS company:

1. Read Write I2C to SPI device grid analyzer.
2. Read Write I2C device RTC.
3. Read Write I2C to UART device ModBus.
4. Read Write I2c to GPIO.
5. Set Gpio to CPLD.
